# Render Static Site Deployment Instructions

1. **Build Command:**
   npm install && npm run build

2. **Publish Directory:**
   dist

3. **Environment Variables:**
   Add all required VITE_ variables (see .env.production)

4. **Root Directory:**
   frontend

5. **For SPA Routing:**
   Render automatically rewrites all routes to index.html for Vite SPAs.

6. **Optional:**
   - For custom domains, add them in the Render dashboard after deployment.
   - If you need SSR, use a Web Service instead of Static Site.

---

**Checklist:**
- [x] package.json scripts: dev, build, preview
- [x] .env.production created
- [x] vite.config.js present
- [x] dist/ will be generated by build
- [x] All code pushed to GitHub

You are ready to deploy!
